データを扱う - 変数と型

プログラムは「データを受け取り、処理して、結果を返す」ことで成り立っています。この章では、プログラムの中でデータを保存・利用するための「変数」と、データの種類を表す「型」の基礎を学びます。「箱」と「ラベル」のたとえを使いながら、図とコード例で丁寧に解説します。

この章で学ぶこと

  • 変数とは何か、宣言・代入・再代入の書き方を理解する
  • 基本データ型（int / double / boolean / char）と参照型（String）の特徴と使い分けを説明できる
  • わかりやすい変数名の命名規則（キャメルケース）を守って変数を宣言できる
  • 算術演算子を使って計算し、結果を画面に表示できる
  • int と double の間で型変換（キャスト）を行える

このページのコード例の読み方

各コード例の右上のラベルで「コードを書く場所」を示します。

  • 完全なプログラムそのまま .java ファイルに貼り付けて実行できます。
  • mainメソッド内に記述public static void main(String[] args) { ... } の中に書いて実行します。

2.1 変数とは — データを入れる「箱」

プログラムで計算や処理をするには、数値・文字・結果などの「データ」をどこかに保存する必要があります。そのための入れ物が変数（へんすう）です。変数は、コンピュータのメモリ上に確保された「名前つきの箱」だとイメージしてください。

flowchart LR A["変数名: age"] --> B["箱の中身: 20"] C["変数名: name"] --> D["箱の中身: 山田太郎"] E["変数名: height"] --> F["箱の中身: 170.5"]

図の説明: 変数名（左）とその箱に格納された値（右）の対応を示しています。

変数を使うには、まず「この名前の箱を用意する」という宣言を行い、次に「箱に値を入れる」代入を行います。Javaでは宣言と代入を1行でまとめて書けます。

Variables.java — 宣言と代入 完全なプログラム 
public class Variables {
    public static void main(String[] args) {
        // 型 変数名 = 値; の形で宣言と代入を同時に行う
        int age = 20;           // 整数型の変数 age に 20 を入れる
        String name = "山田太郎"; // 文字列型の変数 name を宣言

        // 変数の中身を表示する
        System.out.println(age);   // → 20
        System.out.println(name);  // → 山田太郎
    }
}

一度入れた値は再代入で上書きできます。箱の中身を取り替えるイメージです。ただし、型を変えることはできません（後述する型変換を除く）。

Variables.java — 再代入 mainメソッド内に記述 
int score = 80; // 最初は 80 を入れる
System.out.println(score); // → 80

score = 95;     // 再代入：箱の中身を 95 に変える
System.out.println(score); // → 95

用語: 宣言・代入・初期化

「宣言」は変数の箱を用意すること、「代入」は箱に値を入れること、最初の代入（宣言と同時）を「初期化」と呼びます。Javaではローカル変数（メソッド内の変数）は初期化してから使わないとコンパイルエラーになります。

2.2 データ型 — なぜ型があるのか

「なぜ変数には型が必要なのか？」と思うかもしれません。理由は2つあります。1つ目はデータの意味を明確にするため（数値なのか、文字なのかを区別する）、2つ目はメモリを効率よく使うため（整数に必要なメモリと、小数に必要なメモリは異なります）。

Javaには「基本データ型」と「参照型」の2種類があります。初心者がよく使うものをまとめました。

型 分類 入れられる値 使用例
int 基本型 整数（約±21億まで） int age = 20;
double 基本型 小数（浮動小数点数） double pi = 3.14;
boolean 基本型 true または false のみ boolean isOk = true;
char 基本型 1文字（シングルクォート） char grade = 'A';
String 参照型 文字の列（ダブルクォート） String msg = "こんにちは";

int（整数）

年齢・個数・点数など、小数にならない数値に使います。割り算すると小数部は切り捨てられるので注意。

double（小数）

身長・体重・割引率など、小数を扱う場面で使います。精度は約15桁。

boolean（真偽値）

フラグ（ON/OFF）・条件の判定結果に使います。true か false の2値のみ。

String（文字列）

名前・メッセージ・URL など、複数の文字を扱う場面で使います。参照型なので大文字 S で始まります。

char と String の違い

char は1文字のみ（シングルクォート 'A'）、String は0文字以上の文字列（ダブルクォート "Hello"）です。1文字だけを扱う場面は実際には少なく、ほとんどの場面では String を使います。

2.3 変数の命名規則

変数の名前は自由につけられますが、Javaのルールと、チームで通用する慣習（コーディング規則）を守ることが大切です。名前がわかりやすいだけで、プログラムの読みやすさが大きく変わります。

命名ルール（Java標準）

  • 小文字始まりのキャメルケース: 最初の単語は小文字、2語目以降は大文字で始める（例: studentName, totalScore）
  • 使える文字: 英字・数字・アンダースコア _・ドル $（数字で始めることは不可）
  • 予約語は使えない: int, class, if, for など、Javaが特別に使う単語は変数名にできない
  • 意味のある名前: x, a1 より studentAge, itemCount が読みやすい
NamingExample.java — 良い名前と悪い名前 mainメソッド内に記述 
// 悪い例: 意味がわからない
int x = 18;
String s = "田中";

// 良い例: 何を表すか一目でわかる
int studentAge = 18;        // キャメルケース・意味が明確
String studentName = "田中"; // 名詞で何のデータかを示す
boolean isLoggedIn = true;  // boolean は is/has で始めると読みやすい

2.4 演算子と計算

変数に入れた数値を使って計算するための記号を算術演算子と呼びます。数学の記号とほぼ同じですが、掛け算と割り算の記号が異なります。

演算子 意味 例 結果
+ 足し算 10 + 3 13
- 引き算 10 - 3 7
* 掛け算 10 * 3 30
/ 割り算（整数÷整数は切り捨て） 10 / 3 3（3.33…ではない）
% 余り（剰余） 10 % 3 1

注意: 整数同士の割り算は小数が切り捨てられる

10 / 3 の結果は 3.333... ではなく 3 になります。小数の結果が欲しい場合は、どちらかを double にする必要があります（2.5節で詳しく説明します）。

+ 演算子は文字列の連結にも使えます。文字列と別の値を + でつなぐと、すべて文字列として結合されます。

Calculator.java — 計算と文字列連結 mainメソッド内に記述 
int a = 10;
int b = 3;

System.out.println(a + b);  // → 13（足し算）
System.out.println(a - b);  // → 7（引き算）
System.out.println(a * b);  // → 30（掛け算）
System.out.println(a / b);  // → 3（切り捨て！）
System.out.println(a % b);  // → 1（余り）

// 文字列との連結: + は文字列連結になる
String name = "Java";
System.out.println("言語: " + name);    // → 言語: Java
System.out.println("合計: " + (a + b)); // → 合計: 13
terminal — 実行結果 
13
7
30
3
1
言語: Java
合計: 13

2.5 型変換（キャスト）の基本

int の割り算で小数が欲しいときや、double の値を整数として使いたいときに型変換（キャスト）が必要です。

Javaには2種類の型変換があります。「小さい型 → 大きい型」（int → double）は自動で変換されます（暗黙の型変換）。「大きい型 → 小さい型」（double → int）は情報が失われるので、明示的なキャスト（型を丸括弧で書く）が必要です。

CastExample.java — 型変換の例 mainメソッド内に記述 
// 暗黙の型変換: int → double（自動）
int x = 10;
double d = x;          // 10 → 10.0 に自動変換
System.out.println(d); // → 10.0

// 割り算で小数を得る: 片方を double に変換
int a = 10;
int b = 3;
double result = (double) a / b; // a を double にキャスト
System.out.println(result);     // → 3.3333333333333335

// 明示的キャスト: double → int（小数部切り捨て）
double pi = 3.14;
int piInt = (int) pi;   // 小数部が切り捨てられる
System.out.println(piInt); // → 3

型変換の方向性

flowchart LR A["byte"] --> B["short"] --> C["int"] --> D["long"] --> E["float"] --> F["double"] G["自動変換（情報が増える方向）"] H["明示的キャストが必要（情報が失われる方向）"]

図の説明: 左から右へ向かう方向（小さい型から大きい型）は自動変換が可能で、逆方向は情報が失われるため明示的なキャストが必要です。

重要ポイント

キャストは「型を変える」のではなく「その型として見せる」操作です。(int) 3.99 の結果は 4（四捨五入）ではなく 3（切り捨て）になります。四捨五入には Math.round() を使います（後の章で学びます）。

やってみよう 2-1: 四則演算プログラムを作ろう

2つの数値を変数に入れ、足し算・引き算・掛け算・割り算・余りの5つの結果をまとめて表示するプログラムを書いてみましょう。変数名・型・演算子の練習になります。

手順

  • 1 Arithmetic.java という名前のファイルを作り、int numA = 15; と int numB = 4; を宣言する。
  • 2 System.out.println("足し算: " + (numA + numB)); のように、5つの演算結果をラベル付きで表示する。
  • 3 整数の割り算で 3 になることを確認したら、(double) numA / numB で小数の結果も表示してみる。
  • 4 数値を numA = 20; などに変えて再実行し、結果が変わることを確認する。

期待される出力例（numA=15, numB=4）

足し算: 19
引き算: 11
掛け算: 60
割り算（整数）: 3
余り: 3
割り算（小数）: 3.75

理解度確認クイズ

  • Q1. 次のコードの実行結果を答えてください。
    int a = 7; int b = 2; System.out.println(a / b);

    回答を表示

    結果は 3 です。int（整数）同士の割り算では小数部が切り捨てられます。7 ÷ 2 = 3.5 ですが、int では小数を保持できないため 3 になります。3.5 を得るには (double) a / b と書く必要があります。

  • Q2. 次のうち、Javaの変数名として正しいものはどれですか？
    ①1stName　②first_Name　③firstName　④int

    回答を表示

    正しいのは ③ firstName です。① は数字で始まっているためエラー、② はコンパイルできますが慣習（キャメルケース）に反します、④ の int はJavaの予約語なので使えません。Javaの慣習では小文字始まりのキャメルケース（firstName）を使います。

  • Q3. boolean 型の変数に入れられる値は何と何ですか？また、どんな場面で使いますか？

    回答を表示

    boolean 型に入れられる値は true（真） と false（偽） の2種類のみです。「ログインしているか」「メールアドレスが有効か」「スイッチがONかOFFか」のように、2つの状態（フラグ）を管理する場面で使います。次の章で学ぶ条件分岐（if文）とも深く関わります。
