ホーム JavaScript 第9章
第9章 所要時間: 約90分

Webページを動かそう（DOM操作の基礎）

JavaScript を使って Web ページ上の要素を取得し、文字や見た目を変更する方法を学びます。HTML・CSS と JavaScript がどのように連携するのかを理解し、画面を操作する基礎を身につけます。いよいよ「Web操作編」の入り口です。

この章で学ぶこと

  • DOM（ドキュメントオブジェクトモデル）とは何かを説明できる
  • getElementById と querySelector で要素を取得できる
  • textContent で要素のテキストを書き換えられる
  • style プロパティと classList で要素の見た目を変えられる

9.1 DOMとは

ブラウザが HTML ファイルを読み込むと、HTML のタグや文字をもとに「ページの設計図」を内部で自動的に作ります。これを DOM（Document Object Model）（ドキュメントオブジェクトモデル）と呼びます。DOM はページの内容をツリー（木）構造で表現したデータで、JavaScript はこの DOM を通して「ページの要素を取得する・変更する・追加する」といった操作ができます。

HTML・CSS・JavaScript の役割を思い出してみましょう。HTML が「骨組み」、CSS が「デザイン」、JavaScript が「動き」を担当します。JavaScript が「動き」を実現するための窓口が、まさにこの DOM です。JavaScript は DOM を通して、ページ上の任意の要素を読んだり書き換えたりできるのです。

flowchart TD A["HTMLファイル
（ソースコード）"] --> B["ブラウザが読み込む"] B --> C["DOM ツリー
（ページの設計図）"] C --> D["JavaScript で取得・変更"] D --> E["画面に反映される"]

ツリー構造

HTML のタグが親子関係を持った木の枝のような構造になっています。

JavaScriptと連携

JavaScript から DOM にアクセスして、要素の取得・変更・追加ができます。

リアルタイム反映

DOM を変更するとページの表示が即座に更新されます。

DOM ツリーのイメージ

<html> が根（ルート）で、その下に <head> と <body> が枝として伸び、さらにその下に各タグが葉のように広がります。JavaScript は document というオブジェクトを通してこのツリー全体にアクセスできます。

9.2 要素を取得する

DOM を操作するには、まず「操作したい要素を取得する」必要があります。要素を取得する代表的な方法が2つあります。HTML 側に id や class を付けておくことで、JavaScript からその要素を特定できます。

メソッド 対象 HTML側の指定
getElementById("id名") id が一致する1つの要素 id="id名"
querySelector("セレクタ") CSSセレクタに一致する最初の要素 id / class / タグ名など

getElementById は id 属性を使って要素を1つ特定します。ページ内で id は一意（重複なし）にする必要があります。一方、querySelector は CSS のセレクタ（#id名、.クラス名、タグ名 など）を使うため、より柔軟に要素を指定できます。

index.html（HTML側の準備） 
<!-- id 属性を付けておくと JS から取得できる -->
<h1 id="title">こんにちは！</h1>

<!-- class 属性でも取得できる -->
<p class="message">ここはメッセージエリアです。</p>
script.js（要素の取得） 
// id="title" の要素を取得する
const titleEl = document.getElementById("title");
console.log(titleEl); // <h1 id="title">こんにちは！</h1>

// class=".message" の要素を CSS セレクタで取得する
const msgEl = document.querySelector(".message");
console.log(msgEl); // <p class="message">...</p>

// id も querySelector で取得できる（# を付ける）
const titleEl2 = document.querySelector("#title");

要素が見つからない場合は null になる

指定した id や class が HTML に存在しない場合、取得結果は null になります。null に対して操作しようとするとエラーになるため、id / class のスペルミスがないか確認しましょう。

9.3 文字を変える

要素を取得したら、そのテキストの中身を書き換えることができます。使うプロパティは textContent です。代入（=）で新しい文字列をセットすると、ページ上の表示が即座に変わります。

script.js（textContent で文字を書き換える） 
// id="title" の要素を取得
const titleEl = document.getElementById("title");

// textContent を書き換える（画面の文字が変わる）
titleEl.textContent = "JavaScriptで書き換えました！";

// 現在の文字を読み取ることもできる
console.log(titleEl.textContent); // "JavaScriptで書き換えました！"

textContent と innerHTML の違い

textContent は HTML タグをただの文字列として扱い、タグとして解釈しません。一方 innerHTML はタグを HTML として解釈・反映します。基本は textContent を使うほうが安全です（外部からの入力を innerHTML に渡すとセキュリティリスクになる場合があります）。

textContent（安全・推奨）

el.textContent = "<b>太字</b>"
→ タグがそのまま文字として表示

innerHTML（HTML解釈）

el.innerHTML = "<b>太字</b>"
→ 太字として表示される

9.4 見た目を変える

テキストだけでなく、要素の色やサイズなどの見た目も JavaScript から変更できます。方法は大きく2つあります。1つ目は element.style.プロパティ名 で直接スタイルを指定する方法、2つ目は element.classList を使って CSS クラスを付け外しする方法です。

script.js（style プロパティで直接変更） 
const titleEl = document.getElementById("title");

// 文字色を赤に変える
titleEl.style.color = "red";

// フォントサイズを変える（単位も文字列で指定）
titleEl.style.fontSize = "32px";

// 背景色を変える
titleEl.style.backgroundColor = "lightyellow";
script.js（classList でクラスを操作） 
const box = document.querySelector(".box");

// クラスを追加する
box.classList.add("active");

// クラスを削除する
box.classList.remove("active");

// クラスがあれば削除・なければ追加（トグル）
box.classList.toggle("active");

classList を使うほうが実践向き

element.style で直接変更すると、CSS の管理がコードと混在して複雑になります。実際のWeb開発では CSS ファイルにクラスのスタイルを書いておき、JavaScript で classList を使ってクラスを付け外しするパターンが一般的です。「スタイルは CSS、操作は JavaScript」と役割を分けることでコードがすっきりします。

操作 コード例 説明
テキスト変更 el.textContent = "新しい文字" 要素内のテキストを書き換える
スタイル変更 el.style.color = "red" インラインスタイルを直接指定
クラス追加 el.classList.add("active") CSS クラスを追加する
クラス削除 el.classList.remove("active") CSS クラスを削除する
クラス切替 el.classList.toggle("active") クラスをトグル（付け外し）する

やってみよう 9-1: 見出しの文字と色をJavaScriptで変える

HTML ファイルと JavaScript を組み合わせて、ページ上の見出しの文字と文字色を書き換えてみましょう。DOM 操作の基本の流れを体験します。

手順

  1. 1 下の HTML をコピーして practice09.html として保存します。
  2. 2 ブラウザで開き、「元の見出し」と表示されることを確認します。
  3. 3 script タグ内の JavaScript を書き換えて、文字の内容・色・フォントサイズを自由に変えてみましょう。
practice09.html（土台コード） 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>DOM操作 練習</title>
</head>
<body>
  <h1 id="title">元の見出し</h1>
  <p id="desc">ここに説明文が入ります。</p>
  <script>
    // ① 要素を取得する
    const titleEl = document.getElementById("title");
    const descEl = document.getElementById("desc");
    // ② 文字を書き換える
    titleEl.textContent = "JavaScriptで変えました！";
    descEl.textContent = "DOM操作で文字を書き換えています。";
    // ③ 見た目を変える
    titleEl.style.color = "royalblue";
    titleEl.style.fontSize = "36px";
  </script>
</body>
</html>

発展課題

  • querySelector を使って同じ操作をやってみましょう（"#title" のように id を指定）。
  • CSS ファイルに .highlight クラスを定義して、classList.add("highlight") でスタイルを当ててみましょう。

理解度確認クイズ

  1. Q1. DOM とは何ですか？ブラウザと JavaScript の関係で説明してください。

    回答を表示

    DOM（Document Object Model）は、ブラウザが HTML を読み込んで作る「ページの設計図（ツリー構造）」です。JavaScript はこの DOM を通してページ上の要素を取得・変更・追加でき、操作した結果が画面にリアルタイムで反映されます。

  2. Q2. getElementById と querySelector の違いは何ですか？

    回答を表示

    getElementById は id 属性の名前だけを引数に取り、その id を持つ要素を1つ取得します。querySelector は CSS セレクタ（#id・.class・タグ名など）を引数に取り、より柔軟に要素を指定できます。どちらも最初に一致した1つの要素を返します。

  3. Q3. textContent プロパティを使うと、何ができますか？

    回答を表示

    取得した要素のテキストの中身を読み取ったり書き換えたりすることができます。el.textContent = "新しい文字" のように代入すると、ページ上の表示がその文字に変わります。HTML タグを含む文字列を代入してもタグとして解釈されず安全です。
