# JavaScript 入門 学習ガイドライン

このガイドラインでは、プログラミング未経験者・初心者を対象に、JavaScript の基礎を段階的に学習するためのカリキュラムを提供しています。「変数」や「条件分岐」といった基本から、JavaScript の魅力である「Webページを動かす」DOM 操作・イベント処理、そして学んだ知識を組み合わせて作る「ToDoアプリ」までを、身近なたとえと豊富なコード例で丁寧に解説します。プログラミングそのものが初めての方でも、ブラウザ1つで動く JavaScript を通して「自分の書いたコードで画面が動く」楽しさを体験できます。

## 前提条件

### 必要な環境
- パソコン（Windows / macOS / Linux のいずれか）
- インターネット接続（開発ツールのダウンロード用）
- **Webブラウザ**（Google Chrome 推奨。開発者ツールを使います）
- **Visual Studio Code**（学習用エディタ）

### 参考リソース
- [MDN Web Docs - JavaScript（日本語）](https://developer.mozilla.org/ja/docs/Web/JavaScript)
- [JavaScript Primer（日本語の入門書）](https://jsprimer.net/)
- [Visual Studio Code 公式サイト](https://code.visualstudio.com/)
- [Google Chrome 公式サイト](https://www.google.com/chrome/)

### 前提知識
- **必須**: パソコンの基本操作（ファイルの保存、フォルダ作成、アプリのインストール）
- **推奨**: HTML / CSS の基礎知識があると第9章以降の理解がスムーズです（必須ではありません）。プログラミング経験は不要です。

## 学習コンテンツ

### [1. プログラミングとJavaScriptを知ろう](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-01.html)
プログラミングとは何か、JavaScript で何ができるのかを、身近な例を使って分かりやすく説明します。コンピュータと対話する第一歩を踏み出しましょう。

### [2. 開発環境を準備しよう](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-02.html)
プログラムを書くためのエディタ（VSCode）の準備と、ブラウザの開発者ツールの使い方を丁寧に解説します。JavaScript を書いて、実行し、エラーを確認する基本の流れを身につけます。

### [3. はじめてのプログラムを書こう](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-03.html)
「Hello World!」という文字を画面やコンソールに表示する最初のプログラムを作成し、JavaScript の基本的な書き方と実行方法を学びます。

### [4. 変数で情報を記憶しよう](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-04.html)
変数という「箱」を使って、数値や文字列などの情報を記憶・管理する方法を学びます。`let` と `const` の使い分けも、初心者向けに分かりやすく扱います。

### [5. 条件によって処理を変えよう（if文）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-05.html)
「もし〜なら」という条件分岐を使って、状況に応じて異なる処理を行う方法を学びます。比較演算子や真偽値の考え方もあわせて理解します。

### [6. 同じ処理を繰り返そう（for文・while文）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-06.html)
繰り返し処理を使って、同じような処理を効率よく書く方法を学びます。カウント処理や一覧データの処理を通じて、ループの基本を身につけます。

### [7. 処理をまとめよう（関数の基礎）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-07.html)
よく使う処理を関数としてまとめ、必要な時に呼び出して使う方法を学びます。引数、戻り値、関数を使うメリットを、実例を通じて理解します。

### [8. 複数のデータを扱おう（配列とオブジェクト）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-08.html)
複数のデータをまとめて扱う配列と、関連する情報をひとまとまりにできるオブジェクトを学びます。ユーザー情報やタスク情報など、アプリで扱うデータの形をイメージできるようにします。

### [9. Webページを動かそう（DOM操作の基礎）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-09.html)
JavaScript を使って Web ページ上の要素を取得し、文字や見た目を変更する方法を学びます。HTML・CSS と JavaScript がどのように連携するのかを理解し、画面を操作する基礎を身につけます。

### [10. ユーザーの操作に反応しよう（イベント処理）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-10.html)
ボタンクリックや入力操作など、ユーザーの行動に応じて処理を実行する方法を学びます。イベントリスナーの基本を理解し、インタラクティブな Web ページを作れるようになります。

### [11. 入力フォームを扱おう](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-11.html)
テキストボックスやボタンなどのフォーム部品から値を取得し、入力内容に応じて画面を更新する方法を学びます。入力された値を変数や配列に入れて扱う流れを身につけます。

### [12. データを画面に反映しよう（配列とDOMの組み合わせ）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-12.html)
配列に入れたデータをもとに、Web ページへ一覧表示する方法を学びます。データを直接 HTML に書くのではなく、JavaScript で管理して画面に反映する考え方を身につけます。

### [13. 小さな機能を組み合わせよう（ミニアプリ演習）](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-13.html)
DOM 操作、イベント処理、フォーム入力、配列によるデータ管理を組み合わせて、簡単なカウンターアプリやメモ表示アプリを作成します。総合演習に入る前に、複数の知識をつなげて使う練習をします。

### [14. ToDoアプリを作ろう① 画面を作る](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-14.html)
ToDo アプリの土台となる HTML と CSS を作成します。入力欄、追加ボタン、タスク一覧エリアなどを用意し、JavaScript で操作しやすい画面構成を整えます。

### [15. ToDoアプリを作ろう② タスクを追加・表示する](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-15.html)
入力フォームに書いた内容を取得し、タスクとして配列に追加します。配列の内容をもとに画面へ一覧表示することで、「データを変更すると画面も変わる」というアプリ開発の基本的な考え方を学びます。

### [16. ToDoアプリを作ろう③ 完了・削除機能を追加する](https://fcircle-biz.github.io/tech-docs-v2/guide/programming-languages/javascript-ecosystem/javascript/javascript-learning-material-16.html)
タスクの完了状態を切り替える機能と、不要なタスクを削除する機能を実装します。オブジェクトでタスクの状態を管理し、配列の更新に合わせて画面を再表示する流れを身につけます。

## 学習目標

このガイドを完了すると、以下のスキルを身につけることができます：

- JavaScript の開発環境（VSCode・ブラウザ開発者ツール）を自分で準備し、プログラムを作成・実行できる
- 変数・データ型・演算子を使って基本的なデータ処理ができる
- 条件分岐（if）とループ（for / while）でプログラムの流れを制御できる
- 関数を使って処理をまとめ、再利用できる
- 配列とオブジェクトを使って複数のデータをまとめて管理できる
- DOM 操作で Web ページの要素を取得・変更できる
- イベント処理でユーザーの操作に反応するインタラクティブな画面を作れる
- 入力フォームの値を取得し、配列で管理したデータを画面に反映できる
- 学んだ知識を組み合わせて、動作する ToDo アプリを自力で作れる

## 次のステップ

このガイドで JavaScript の基礎を習得したら、以下のような発展的なトピックへ進むことをおすすめします。

- **モダン JavaScript**: ES Modules、非同期処理（Promise / async-await）、fetch によるデータ取得
- **TypeScript**: 型を付けて安全に大規模開発を行う
- **フロントエンドフレームワーク**: React / Vue / Next.js による本格的な Web アプリ開発
- **Node.js**: サーバーサイド JavaScript とバックエンド開発
