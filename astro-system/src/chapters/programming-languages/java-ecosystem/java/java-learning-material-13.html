便利な道具を使いこなす - Java標準ライブラリ

Javaには最初から便利な機能（標準ライブラリ）が大量に用意されています。この章では「全部自分で作らなくてもいい」という考え方を学び、文字列操作・数学計算・日付と時間など、実際のプログラムで毎日使う機能を手を動かしながらマスターしましょう。

この章で学ぶこと

  • 標準ライブラリとは何か、import の役割を説明できる
  • String クラスの代表メソッド（length / substring / replace / split など）を使いこなせる
  • Math クラスで数値計算（絶対値・べき乗・乱数など）を行える
  • java.time の LocalDate / LocalDateTime で日付を取得・操作・表示できる
  • Random や StringBuilder など補助クラスの使いどころを理解できる

13.1 標準ライブラリとは

プログラムを書いていると「文字の長さを調べたい」「今日の日付を取得したい」「ランダムな数が欲しい」といった場面が何度も出てきます。こうしたよく使う機能は、Javaに最初から用意されています。これを標準ライブラリと呼びます。

料理にたとえると、標準ライブラリは「よく使う調理道具が全部揃ったキッチン」のようなものです。包丁やフライパンを自分で鍛冶屋に頼まなくても、最初から棚に並んでいます。プログラマーは「どの道具を使うか選ぶだけ」で済みます。

import とは

標準ライブラリのクラスを使うには、ファイルの先頭で import を書きます。「この道具棚を使う」と宣言するイメージです。

java.lang は import 不要

String・Math・System など java.lang パッケージのクラスは自動的に使えます。import を書かなくてもOKです。一方、java.time.LocalDate・java.util.Random など他のパッケージは import が必要です。

ImportExample.java — import の書き方 
// java.lang（String, Math など）は import 不要
// それ以外のパッケージは先頭で import する
import java.time.LocalDate;   // 日付クラス
import java.util.Random;      // 乱数クラス

public class ImportExample {
    public static void main(String[] args) {
        // String は import なしで使える
        String name = "Java";
        // LocalDate は import が必要
        LocalDate today = LocalDate.now();
    }
}

13.2 文字列操作（String クラス）

String クラスは、文字列を操作するためのメソッドを多数持っています。プログラムで文字を扱う場面は非常に多いため、代表的なメソッドを覚えておくと一気に作れるものが増えます。

メソッド 説明 例
length() 文字数を返す "Java".length() → 4
substring(開始, 終了) 部分文字列を切り出す "Hello".substring(1,3) → "el"
toUpperCase() 大文字に変換 "hello".toUpperCase() → "HELLO"
toLowerCase() 小文字に変換 "JAVA".toLowerCase() → "java"
replace(旧, 新) 文字を置き換える "abc".replace("b","B") → "aBc"
split(区切り文字) 文字列を配列に分割 "a,b,c".split(",") → ["a","b","c"]
contains(文字列) 含まれているか調べる "Hello".contains("ell") → true
equals(文字列) 文字列が等しいか比較 "abc".equals("abc") → true

注意: 文字列の比較は == ではなく equals() を使う

Javaで文字列を比較するとき、== は「同じオブジェクトか」を調べます。文字の内容が等しいかどうかを調べるには、必ず equals() を使いましょう。

StringMethods.java — 代表メソッドの実演 
public class StringMethods {
    public static void main(String[] args) {
        String text = "Hello, Java!";

        // 文字数を調べる
        System.out.println(text.length());       // → 12

        // 先頭から5文字切り出す（0〜4番目）
        System.out.println(text.substring(0, 5)); // → Hello

        // 大文字に変換する
        System.out.println(text.toUpperCase());   // → HELLO, JAVA!

        // 「Java」を「World」に置き換える
        System.out.println(text.replace("Java", "World")); // → Hello, World!

        // カンマで分割する
        String csv = "apple,banana,cherry";
        String[] fruits = csv.split(",");
        for (String f : fruits) {
            System.out.println(f); // apple / banana / cherry
        }
    }
}

13.3 数学計算（Math クラス）

Math クラスは数学的な計算をサポートします。java.lang パッケージに含まれるため import は不要です。すべてのメソッドは Math.メソッド名() の形で呼び出します。

メソッド 説明 例
Math.max(a, b) 大きい方の値を返す Math.max(3, 7) → 7
Math.min(a, b) 小さい方の値を返す Math.min(3, 7) → 3
Math.abs(a) 絶対値を返す Math.abs(-5) → 5
Math.pow(x, n) x の n 乗 Math.pow(2, 10) → 1024.0
Math.sqrt(x) 平方根 Math.sqrt(9) → 3.0
Math.round(x) 四捨五入 Math.round(3.6) → 4
Math.random() 0.0〜1.0 未満の乱数 Math.random() → 0.735...
MathExample.java — Math クラスを使う 
public class MathExample {
    public static void main(String[] args) {
        // 大きい方を選ぶ
        System.out.println(Math.max(10, 25));   // → 25

        // 絶対値（マイナスをプラスにする）
        System.out.println(Math.abs(-42));       // → 42

        // 2の10乗（1024）
        System.out.println((int) Math.pow(2, 10)); // → 1024

        // 平方根
        System.out.println(Math.sqrt(16));       // → 4.0

        // 0〜99 のランダムな整数を作る
        int rand = (int)(Math.random() * 100);
        System.out.println("乱数: " + rand);
    }
}

13.4 日付と時間（java.time）

Java 8 から追加された java.time パッケージは、日付・時間を扱う現代的なAPIです。使いやすく、バグが起きにくい設計になっています。よく使う2つのクラスを覚えましょう。

LocalDate

年・月・日だけを扱うクラス。時刻は含まない。「今日は何日？」に使う。

LocalDateTime

年・月・日・時・分・秒を扱うクラス。「今は何時何分？」まで知りたいときに使う。

DateExample.java — 日付の取得と操作 
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class DateExample {
    public static void main(String[] args) {
        // 今日の日付を取得する
        LocalDate today = LocalDate.now();
        System.out.println("今日: " + today); // → 2026-06-18

        // 1週間後の日付を計算する
        LocalDate nextWeek = today.plusDays(7);
        System.out.println("1週間後: " + nextWeek); // → 2026-06-25

        // 現在の日時を取得してフォーマットする
        LocalDateTime now = LocalDateTime.now();
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("yyyy年MM月dd日 HH:mm");
        System.out.println(now.format(fmt)); // → 2026年06月18日 10:30
    }
}

重要ポイント: 日付オブジェクトは変更されない

plusDays() は元のオブジェクトを書き換えません。新しい日付オブジェクトを返します。LocalDate nextWeek = today.plusDays(7); のように必ず変数に受け取りましょう。

13.5 その他の便利クラス（Random ＆ StringBuilder）

標準ライブラリにはまだまだ便利なクラスがあります。ここでは実践でよく登場する2つを紹介します。

Random クラス

Math.random() より扱いやすく、整数の乱数が直接取れます。ゲームやテストデータ生成に便利です。

RandomExample.java 
import java.util.Random;

public class RandomExample {
    public static void main(String[] args) {
        Random random = new Random();

        // 0〜9 のランダムな整数
        int num = random.nextInt(10);
        System.out.println("乱数: " + num);

        // true か false をランダムに得る
        boolean coin = random.nextBoolean();
        System.out.println("コイン: " + coin);
    }
}

StringBuilder クラス

文字列を何度も連結するとき、+ を繰り返すより StringBuilder を使う方が効率的です。append() で追記し、最後に toString() で String に変換します。

StringBuilderExample.java 
public class StringBuilderExample {
    public static void main(String[] args) {
        // 1〜5 を文字列として連結する
        StringBuilder sb = new StringBuilder();
        for (int i = 1; i <= 5; i++) {
            sb.append(i);      // 数字を追加
            if (i < 5) sb.append(", "); // 区切り文字
        }
        // StringBuilder を String に変換して表示
        System.out.println(sb.toString()); // → 1, 2, 3, 4, 5
    }
}

ライブラリの探し方

「こんな機能がほしい」と思ったら、まず「Java 標準ライブラリ ○○」で検索してみましょう。Java の公式ドキュメント（Java API ドキュメント）にクラスとメソッドの一覧があります。すでに用意されていることがほとんどです。

やってみよう 13-1: 今日と1週間後の日付を表示しよう

標準ライブラリを複数組み合わせて、今日の日付と1週間後の日付を表示するプログラムを作りましょう。さらに余裕があれば、文字列を分割して単語ごとに表示する機能も追加してみてください。

課題①: 日付表示

  1. 1 LocalDate.now() で今日の日付を取得して表示する。
  2. 2 plusDays(7) で1週間後の日付を計算して表示する。
  3. 3 DateTimeFormatter で「yyyy年MM月dd日」の形式にフォーマットして表示する。

課題②（発展）: 単語分割

  1. 1 "apple banana cherry" という文字列を split(" ") でスペース区切りに分割する。
  2. 2 for 文でそれぞれの単語を1行ずつ大文字で表示する（toUpperCase() を使う）。

理解度確認クイズ

  1. Q1. String クラスで文字列が等しいかどうかを比較するには、どのメソッドを使いますか？また、== ではなくそのメソッドを使う理由を説明してください。

    回答を表示

    equals() メソッドを使います。== は「同じオブジェクトを指しているか」を調べるため、内容が同じ文字列でも false になることがあります。文字の内容を比較するには equals() を使う必要があります（例: "abc".equals("abc") → true）。

  2. Q2. Math.pow(2, 8) の結果は何ですか？また、結果の型は何型ですか？

    回答を表示

    結果は 256.0 で、型は double 型です。Math.pow() は常に double を返します。整数として扱いたい場合は (int) Math.pow(2, 8) のようにキャストします。

  3. Q3. java.time.LocalDate を使うとき、なぜ import 文が必要なのですか？逆に import が不要なパッケージはどれですか？

    回答を表示

    java.time.LocalDate は java.time パッケージに属しているため、使う前に import java.time.LocalDate; と宣言が必要です。一方、java.lang パッケージ（String・Math・System など）はJavaが自動的に読み込むため、import を書かなくても使えます。
