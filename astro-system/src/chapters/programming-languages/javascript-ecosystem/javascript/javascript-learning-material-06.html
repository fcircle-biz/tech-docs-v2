同じ処理を繰り返そう（for文・while文）

繰り返し処理（ループ）を使って、同じような作業を効率よくこなす方法を学びます。「1から100まで足す」「リストの全項目を表示する」など、プログラムの核心ともいえるループの仕組みを、for文とwhile文の2つのアプローチで丁寧に解説します。

この章で学ぶこと

  • 繰り返し処理（ループ）がなぜ必要なのかを理解する
  • for文の3つの部品（初期化・条件・更新）を正しく書ける
  • while文の書き方と、for文との使い分けがわかる
  • 合計計算や配列の全要素表示など、ループの実用的な使い方を身につける

6.1 繰り返しとは — なぜループが必要か

「1から10の数を順番にコンソールに表示する」プログラムを書くとしたら、どうしますか? console.log(1)、console.log(2)…と10行書けば確かに動きます。でも1000件のデータを表示したいとき、同じように1000行書くのは現実的ではありません。

そこで登場するのが繰り返し処理（ループ）です。ループを使えば「同じ処理を何度も繰り返す」という指示を、短いコードで表現できます。同じ作業を繰り返すのは人間よりコンピュータの得意分野です。ループはプログラミングの最も強力な道具の一つです。

コードが短くなる

何千回繰り返す処理でも、数行のループで書けます。

変更に強い

「10回」を「100回」に変えるときも、数字を1か所直すだけです。

データ処理に必須

配列の全要素を処理するなど、実際のアプリで欠かせない技術です。

6.2 for文 — 回数が決まっているループ

for文は、繰り返す回数があらかじめ決まっているときに使います。構文は次のとおりです。

for文の基本構文 
// for ( 初期化 ; 条件 ; 更新 ) { 処理 }
for (let i = 0; i < 5; i++) {
    console.log(i); // 0, 1, 2, 3, 4 と表示される
}

カッコの中には3つの部品がセミコロン（;）で区切られて並んでいます。それぞれの役割を一つずつ見ていきましょう。

部品 例 役割 実行タイミング
初期化 let i = 0 カウンタ変数を用意する ループ開始前に1回だけ
条件 i < 5 trueの間ループを続ける 各ループの実行前に毎回
更新 i++ カウンタを1増やす 各ループの処理後に毎回

for文がどのように動くかを図で確認しましょう。

flowchart TD A["① 初期化
let i = 0"] --> B{"② 条件チェック
i < 5 ?"} B -->|"true（続ける）"| C["③ 処理を実行
console.log(i)"] C --> D["④ 更新
i++"] D --> B B -->|"false（終了）"| E["ループ終了"]

具体的な例を動かして確認してみましょう。

for-loop-example.js 
// 1から5まで順番に表示する
for (let i = 1; i <= 5; i++) {
    console.log("カウント: " + i);
}
// 出力: カウント: 1
//       カウント: 2
//       カウント: 3
//       カウント: 4
//       カウント: 5

カウンタ変数 i について

i という名前は「index（インデックス＝番号）」の略で、ループのカウンタ変数の慣習的な名前です。j、k、count など別の名前でも構いませんが、ループで i を見かけたら「ループのカウンタ」だとすぐわかります。

6.3 while文 — 条件が変わるまで続けるループ

while文は「条件がtrueである限り、処理を繰り返す」という構文です。繰り返す回数を最初から決めず、「ある条件が満たされるまで続ける」ときに向いています。

while-loop-example.js 
// 5からカウントダウンする
let count = 5;

while (count > 0) {
    console.log(count + "秒前...");
    count--; // カウンタを1減らす（必須！）
}

console.log("発射！");
// 出力: 5秒前... 4秒前... 3秒前... 2秒前... 1秒前... 発射！

注意: 無限ループに気をつけよう

while文の中でカウンタ変数を更新し忘れると、条件が永遠にtrueのままになり「無限ループ」が発生します。ブラウザが固まる原因になるため注意が必要です。

対策: ループ内で必ず条件を変化させる処理（count-- や i++ など）を書きましょう。

for文とwhile文の使い分け

使う文 こんなとき 例
for文 繰り返す回数が最初からわかっている 「5回繰り返す」「配列の全要素を処理する」
while文 回数は決まっていないが、条件で終了を判断する 「ユーザーが正しい値を入力するまで」「残りが0になるまで」

迷ったらfor文を選ぼう

実際のプログラムでは、for文のほうが圧倒的に使われます。「回数がわかるならfor、条件次第ならwhile」と覚えておけば、ほとんどの場面で適切な選択ができます。

6.4 ループの活用例

ループが実際どんな場面で役立つか、具体的な例で確認しましょう。

合計を計算する

sum-example.js 
// 1から10の合計を計算する
let total = 0; // 合計を入れる変数（0からスタート）

for (let i = 1; i <= 10; i++) {
    total = total + i; // iの値を合計に足していく
}

console.log("1から10の合計: " + total); // → 55

配列の全要素を表示する（予告）

配列（複数のデータをまとめたもの）は第8章で詳しく学びますが、ループとの組み合わせが非常に重要なので、ここで軽く触れておきます。

array-loop-example.js 
// 配列の全員の名前を表示する
const names = ["Alice", "Bob", "Carol"];

for (let i = 0; i < names.length; i++) {
    console.log("こんにちは、" + names[i] + "さん！");
}
// → こんにちは、Aliceさん！
// → こんにちは、Bobさん！
// → こんにちは、Carolさん！

break と continue — ループを途中でコントロールする

break — ループを即座に終了

「この条件になったらループをやめる」ときに使います。

for (let i = 1; i <= 10; i++) {
  if (i === 5) break; // 5でやめる
  console.log(i); // 1,2,3,4
}

continue — 今回の処理をスキップ

「この回だけ飛ばして次に進む」ときに使います。

for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3をスキップ
  console.log(i); // 1,2,4,5
}

重要ポイント

ループは「同じ形の処理を自動化する」最も基本的な仕組みです。for文・while文・break・continueをマスターすると、データの集計や一覧表示など、実際のアプリで必要な処理がほとんど書けるようになります。次章の「関数」と組み合わせると、さらに強力なコードが書けます。

やってみよう 6-1: ループを使った計算と表示

以下の2つの課題のどちらかに挑戦しましょう。ブラウザの開発者ツール（コンソール）を開いて、コードを実際に入力・実行してみてください。

課題A: 1から100の合計を計算する

  1. 1 let total = 0; で合計用の変数を用意する。
  2. 2 for文で i を 1 から 100 まで増やしながら total += i; を実行する。
  3. 3 ループ後に console.log(total); で結果を表示する。答えは 5050 になるはずです！

課題B: 好きな段の九九を表示する

  1. 1 const dan = 7; のように好きな段を変数に入れる。
  2. 2 for文で i を 1 から 9 まで増やし、console.log(dan + " × " + i + " = " + (dan * i)); を実行する。
  3. 3 7の段なら「7 × 1 = 7」から「7 × 9 = 63」まで表示されることを確認する。

理解度確認クイズ

  1. Q1. for (let i = 0; i < 3; i++) のカッコ内の3つの部品の名前と役割を答えてください。

    回答を表示

    ①初期化（let i = 0）: カウンタ変数を0に設定する。ループ開始前に1回だけ実行。②条件（i < 3）: trueの間ループを続ける。毎回の実行前にチェック。③更新（i++）: カウンタを1増やす。毎回の処理後に実行。

  2. Q2. for文とwhile文はどのように使い分けると良いでしょうか？

    回答を表示

    for文は繰り返す回数が最初からわかっているときに使います（例: 「5回繰り返す」「配列の全要素を処理する」）。while文は回数は決まっていないが、ある条件が満たされるまで続けたいときに使います（例: 「正しい入力があるまで」）。迷ったら先にfor文を検討しましょう。

  3. Q3. 次のコードは何を出力しますか？ for (let i = 1; i <= 4; i++) { if (i === 3) continue; console.log(i); }

    回答を表示

    出力は 1, 2, 4 の3つです。continue は「今回の処理をスキップして次のループへ進む」命令なので、i === 3（i が3）のときだけ console.log が実行されずスキップされます。
