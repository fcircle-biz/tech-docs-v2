# Claude Code学習教材

Anthropic 公式のエージェント型コーディング CLI「Claude Code」を、基礎から実践まで段階的に学べる入門学習ガイド（全14章）です。AI 支援開発の基礎知識から、インストール・基本操作・コードベース探索・メモリ管理・Git 連携・MCP 連携・サブエージェント自動化・実践ワークフローまでを順序立てて解説します。

| 項目 | 内容 |
|------|------|
| 難易度 | 初級 |
| 所要時間 | 約16時間 |
| 分類 | 開発手法・プロセス |
